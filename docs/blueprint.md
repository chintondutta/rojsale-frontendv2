# **App Name**: Roje Admin

## Core Features:

- Dashboard: Display widgets for key metrics (new users, expenses, ads, packages) with animated counters and sparklines; also displays bar and donut charts visualizing relevant data.
- User Management: Enable CRUD operations (create, read, update, delete) for user accounts, featuring search, export, and pagination.
- Ad Management: Enable CRUD operations and filtering, sorting, and tooltips to provide detailed ad property information.
- Category Management: Enable CRUD operations with icon previews and badge counts, enhancing visual representation.
- Package Management: Enable CRUD operations with quick views and feature badges, displaying package attributes effectively.
- Payment Tracking: Enable payment tracking via a table with status badges, expandable rows, exports, and filters for efficient transaction oversight.
- Role Based Access Control: Enable permissions configured based on roles and LLM assisted tool that warns the user about unintended escalation of permission.

## Style Guidelines:

- Primary color: Deep Blue (#5a5cf5) for main UI elements, sidebar, and buttons to maintain brand consistency.
- Accent color: Bright Yellow (#ffe800) for active menu items and highlights, providing visual cues.
- Success color: Light Green Tint (#dcebe0) for positive feedback messages and status indications.
- Card Background: White (#ffffff) for all data cards and modals, ensuring readability.
- Text Color: Dark Gray (#333333) for main text to ensure legibility and contrast.
- Page Background: Pale Green for layering and to provide visual depth to the layout.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, neutral look that is suitable for both headlines and body text.
- Utilize icons from Material or Lucide libraries throughout the dashboard, adding visual cues and improving user navigation.
- Maintain consistent spacing (p-6 for cards, m-4 for page/section gaps) for a structured, compact UI design.
- Apply rounded corners (rounded-lg) and soft shadows (shadow-md) to UI elements, enhancing visual depth and modern feel.
- Incorporate smooth transition animations (transition-all duration-200) on hover states, button clicks, and table interactions, improving overall UX.
